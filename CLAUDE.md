## 第3層：実務ルール

### プロジェクト情報

- **公開URL**: https://murinaikurashi.com/apps/circle-of-fifths/
- **リポジトリ**: https://github.com/daikainozaki-cyber/circle-of-fifths
- **技術スタック**: HTML/CSS/JavaScript（Pure）、SVG、Xserver

### デプロイ

**Github Actions自動デプロイ**
```bash
git add . && git commit -m "変更内容" && git push
```
→ mainブランチへのpushで自動的にXserverに反映

**手動デプロイ（緊急時）**
```bash
./deploy.sh
```

### ファイル構成

| ファイル | 説明 |
|---------|------|
| `index.html` | アプリ本体（すべて1ファイル） |
| `deploy.sh` | 手動デプロイスクリプト |
| `README.md` | 詳細ドキュメント・更新履歴 |
